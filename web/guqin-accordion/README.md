# 古琴 → 手风琴 · Guqin Accordion Generator

Displays famous Chinese guqin pieces as sheet music and transcribes them for accordion — with a synthetic accordion preview using Tone.js.

## Features

- **4 famous pieces**: 阳关三叠, 梅花三弄, 酒狂, 广陵散
- **Two views**: original guqin notation and accordion adaptation (grand staff)
- **Audio preview**: synthetic accordion sound (fat sawtooth + tremolo + reverb)
- **Adjustable tempo** and volume
- Sheet music rendered by [abcjs](https://paulrosen.github.io/abcjs/)

## Run locally

```sh
npm install
npm run dev   # dev server at localhost:5173
npm run build
```
