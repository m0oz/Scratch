# 日本語 Lernen

Japanese vocabulary learning app with spaced repetition and speaking practice.

## Features

- **Flashcards** with flip animation (Japanese → German)
- **Spaced Repetition (SRS)** — SM-2-based algorithm schedules reviews automatically
- **Text-to-Speech** — hear correct Japanese pronunciation with one tap
- **Speech Recognition** — practice speaking; app checks your pronunciation
- **7 built-in decks** — Greetings, Numbers, Daily Verbs, Adjectives, Food, Useful Phrases, Time & Date (~105 cards)
- **Custom cards/decks** — add your own vocabulary, stored in localStorage

## Running locally

```sh
npm install
npm run dev   # → localhost:5173
```

## Stack

React + TypeScript + Tailwind CSS + Vite. No external dependencies beyond the template.
