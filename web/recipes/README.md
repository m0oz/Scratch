# Recipes

A mobile-first recipe manager. Store, browse, and scale your recipes.

**Live:** `https://m0oz.github.io/Scratch/web/recipes/`

## Features

- Add, edit, and delete recipes
- Search by name/description
- Filter by category (Breakfast, Lunch, Dinner, etc.)
- Servings scaler — adjust servings and ingredient amounts update automatically
- Prep/cook time tracking
- Starts with 3 sample recipes; all data saved to localStorage

## Run locally

```bash
cd web/recipes
npm install
npm run dev
```
