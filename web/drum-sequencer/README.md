# DR-808 Drum Sequencer

A browser-based drum machine inspired by the Roland TR-808, with a 16-step sequencer and synthesized drum sounds.

## Features

- **10 drum sounds** — kick, snare, closed/open hi-hat, clap, tom, rimshot, cowbell, crash, ride — all synthesized via Web Audio API (no samples needed)
- **16-step sequencer** with visual playhead and beat markers
- **4 preset patterns** — Amen Break, Four on the Floor, Boom Bap, Dembow
- **Adjustable tempo** (60–200 BPM) and **swing**
- **Click track names** to preview individual sounds
- Retro 80s drum machine UI with LED displays and glow effects

## Run locally

```sh
npm install
npm run dev
```

Opens at `http://localhost:5173`.
